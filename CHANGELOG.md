# Changelog

Все значимые изменения проекта фиксируются в этом файле.

## [1.0.0] - 2026-04-06

### Added
- Полный двухэтапный пайплайн: детекция кассы + классификация состояния.
- Скрипт подготовки датасета `prepare_dataset.py` с поддержкой публичного Roboflow URL без API-ключа.
- Обучение на GPU и сохранение лучших весов моделей.
- Проверка без утечки данных: demo-изображения отделены от train.
- GUI для демонстрации инференса на отложенных изображениях.
- Документация проекта (`README.md`), зависимости (`requirements.txt`) и правила игнорирования (`.gitignore`).

### Notes
- Проект ориентирован на учебную задачу и может требовать расширения датасета для более стабильной работы в новых условиях съёмки.
